Water Beacons

Water Beacons is a resilient communication system designed for disaster scenarios like floods, where conventional cellular networks often fail. The system creates an independent, ad-hoc mesh network using low-power radio modules (LoRa).

Technology Used:

  • LoRa (Long Range) communication modules for data transmission over several kilometers.
  • ESP32 microcontrollers to manage the network nodes.
  • A custom mesh networking protocol to ensure messages can hop between beacons to reach their destination.

Smart MicroLab

Smart MicroLab is a portable, AI-powered diagnostic tool that assists in the rapid detection of critical diseases. It leverages machine learning to analyze microscopic images of samples (like blood smears) to identify pathogens or cellular abnormalities.

Core Functionality:

  • A compact hardware setup with a digital microscope connected to a single-board computer.
  • A Convolutional Neural Network (CNN) model trained to classify and detect specific diseases.
  • An intuitive user interface for capturing images and viewing the AI-generated analysis.
